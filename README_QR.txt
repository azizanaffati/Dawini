DAWINI static website

Files:
- index.html
- styles.css
- script.js

How to open locally:
1. Put the 3 files in the same folder.
2. Double click index.html.

Important for QR code:
A QR code must point to a public URL if you want it to open on a phone.
So after creating the static site, upload it to one of these:
- GitHub Pages
- Netlify
- Vercel

Then generate a QR code from the final public link.
